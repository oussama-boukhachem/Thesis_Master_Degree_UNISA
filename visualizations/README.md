# Visualization and Analysis Charts

This directory contains the graphical plots generated by the `visualize_results.py` Python script. These charts translate the complex confidence intervals and distributions from the Hierarchical Bootstrap analysis into accessible visual summaries, directly supporting the core findings of the thesis.

## Chart Overviews

The output images represent different analytical viewpoints comparing the Handwritten benchmarks against their AI-Generated counterparts:

* **`01_overall_dashboard.png`**: A comprehensive master dashboard summarizing the entire experimental outcome. It combines regression detection success rates (A), benchmark coverage distribution by commit (B), categorical performance impact box-plots (C), and summarized numerical statistics (D) into one readable view.

* **`02_detection_rate.png`**: A straightforward bar chart demonstrating the core finding of the experiment: both human-written and AI-generated benchmarks achieved a 100% success rate in detecting the injected regressions with statistical significance.

* **`03_impact_by_commit.png`**: A detailed, side-by-side performance degradation chart split across the three Apache Ignite commits (v1.8.0, v2.0.0, and v2.2.0). It charts each benchmark method with its calculated 95% Confidence Interval error bars on a logarithmic scale.

* **`04_impact_by_category.png`**: Box plot graphs showing how different types of Java operations (Cache, Future/Async, and Atomic Increments) responded to the CPU-consuming injection. It highlights that the AI-generated benchmarks scaled identically to handwritten benchmarks within the same operation category.

* **`05_impact_distribution_comparison.png`**: A comparative side-by-side view combining the categorical impact distribution box plots with a mean-average bar chart. Note: The y-axis scales are extremely large because the `Blackhole.consumeCPU(1000000L)` injection caused regressions mathematically spanning thousands of percentage points over the baseline.

* **`06_all_benchmarks_ranked.png`**: A horizontal bar chart ranking every single benchmark (grouped by AI vs Handwritten) from lowest performance impact to highest. This clearly visualizes how closely the AI twin benchmarks mirror the specific behavior of their human counterparts.

* **`07_summary_table.png`**: A rendered table providing the exact numerical limits, medians, and detection rates across all three commits, proving the statistical equivalence between the two sets.

* **`08_benchmark_coverage.png`**: A simple bar chart validating the symmetry of the experiment, confirming that exactly 2, 7, and 7 valid benchmark pairs were tested across the three commits respectively.

## Generation
These visualizations can be regenerated exactly as they appear here by running:
```bash
python scripts/visualize_results.py
```
*(Ensure the statistical data in `results/` is present before running the python script).*
